Universal Staff-Level Principles
Amounts in integer cents — NEVER float
0.1 + 0.2 ≠ 0.3 in IEEE 754. Store as BIGINT. $100.00 = 10000. Satoshis as BIGINT. Never DECIMAL(x,y) for money arithmetic.
Write log BEFORE external call
Log psp_requests / blockchain_tx as PENDING before making any external call. Enables crash recovery. Without this: timeout = unrecoverable state.
Timeout ≠ Failure — UNKNOWN state is real
Never assume failure on timeout. Set UNKNOWN state. Async reconciliation job queries authoritative source. Never retry without querying first.
Immutable audit trail — append only
Ledger entries, payment events, kyc_records never updated or deleted. Corrections create new entries. Essential for regulatory compliance.
Idempotency at EVERY layer
Gateway → Service → DB → PSP/Blockchain calls → Webhooks. INSERT ON CONFLICT DO NOTHING at DB level as final backstop. Safe retries everywhere.
Optimistic over pessimistic locking
UPDATE WHERE status='EXPECTED', check rows_affected=1. No lock held between read and write. Kills throughput if you use SELECT FOR UPDATE at scale.
Saga replaces 2PC always
2PC blocks entire transaction if any participant is unavailable. Saga + compensating transactions: each step independent, rollback via explicit reversal.
Partial index on active subset
WHERE status NOT IN ('SETTLED','COMPLETE'). 99.9% of rows are terminal — indexing them wastes space and slows writes. Active subset is what ops query constantly.
